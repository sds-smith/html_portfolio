Portfolio

Cup

A social media application for coffee lovers built on the AT Protocol (the same technology powering Bluesky).

CUP allows authenticated users to create posts (called check-ins) about the coffee they are drinking, detailing the coffee name, roast level, brewing method, and more. Users can view a federated feed of check-ins from other coffee obsessives.

Liquid Glass UI

A React component library for building interfaces with translucent, depth-aware surfaces inspired by the Liquid Glass design language. Built on Radix UI primitives, featuring zero-runtime CSS generation with CSS Variables and CSS Modules, and bundled with tsup.

Discover Breweries

A brewery locator app built with React, TypeScript, and serverless AWS Lambda functions. Powered by the OpenBreweryDB API and MongoDB.

Assemble the Jams

A playlist generator Express app with React frontend, powered by the Spotify API.
